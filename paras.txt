Empire State Building
No New York vacation would be complete without a visit to the Empire State Building. Ride to the top of the 86th-floor observation deck and take in the views with the New York Pass. 

                    Free Entry and Skip the Line to The Empire State Building Normally $39.00 - Free with New York Pass











Overview
Opening Times
Getting there
Attractions Nearby







Synonymous with New York City, the Empire State Building is an American icon and one of the Seven Wonders of the Modern World. Head to the top and enjoy views of up to 80 miles on a clear day, taking in Central Park, Manhattan, New Jersey, Connecticut and even out to Pennsylvania and Massachusetts.
Located in Midtown Manhattan, the views from its 86th-floor observation decks are unmissable. No need for individual Empire State Building tickets - it's all included with the pass.
New York Pass Tips

Fast-track entry for New York Pass holders - head straight to security, no need to collect a ticket from the kiosks or box office
Beat the crowds: for more space and less waiting, plan your visit between 08.00 - 11.00
The Empire State Building have unveiled a brand new entrance on 34th street, just for visitors to the Observatory. This larger entrance will help create a seamless flow and provide a fully modernized journey for all visitors.
There are two observation decks: open-air on the 86th-floor included with the New York Pass, and indoors on the 102nd-floor (upgrade to be purchased at the Empire State Building kiosk)
Don't miss the Empire State Building history exhibition
Open until 02.00 – enjoy late-night entry

Did You Know? Empire State Building Facts

It took one year and 45 days to build
How tall is the Empire State Building? 1454 feet.
Four million people visit the Empire State Building every year
It has its own zip code: 10118
Valentine’s Day is the only day that couples can get married on top of the Empire State Building

  function createCookie(name, value, days) {
   var expires;
   if (days) {
    var date = new Date();
    date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
    expires = '; expires=' + date.toGMTString();
   }
   else {
    expires = '';
   }
   document.cookie = encodeURIComponent(name) + "=" + encodeURIComponent(value) + expires + '; path=/';
  }

  function readCookie(name) {
   var keyValue = document.cookie.match('(^|;) ?' + name + '=([^;]*)(;|$)');
   return keyValue ? keyValue[2] : null;
  }
